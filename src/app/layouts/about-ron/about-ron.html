About the Ron Language

The Ron language (also known as Challa, or Chala) is an indigenous language cluster native to Plateau State, Nigeria. It is classified as an Afro-Asiatic language.


Key Status & Classification

Native to Nigeria (Plateau State)
Estimated Speakers 230,000 (as of 2020)
Language Vitality Status Stable. The language is the norm in the home and community, with all children learning and using it.
Classification Afro-Asiatic · Chadic · West · Bole–Angas · Ron (A.4)

Language Varieties (Dialects)

The Ron cluster comprises several distinct varieties, which some linguists consider separate languages:

  1. Bokkos
  2. Mbar
  3. Daffo-Butura
  4. Shagawu
  5. Maleni
  6. Manguna
  7. Nafunfia

Historical Note

While modern Ron uses a decimal (base 10) counting system, historical evidence indicates that the language previously employed a duodecimal (base 12) counting system.
